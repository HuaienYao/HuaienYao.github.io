前言

总感觉LeaveIt这个黑白主题的选中文字的颜色和背景颜色怪怪的，因为是浅蓝色的，有点和黑白不搭。所以就把修改选中文字的颜色和背景颜色的过程也记录下。

修改方法

因为是样式，就是修改主题下的css文件了。打开/themes/LeaveIt/assets/css/_common/_core/base.scss Ctrl+F 搜索::selection 定位或者找到第43~45行*（这是默认未修改的文件行数）

  ::selection {
    background: rgba(0, 149, 255, 0.1);
  }

修改为以下内容，增加的

  ::selection {
    background: #080808;
	color: #FFFFFF;
  }
  ::-moz-selection {
    background: #080808;
    color: #FFFFFF;
}
  ::-webkit-selection {
    background: #080808;
    color: #FFFFFF;
}

保存后用hugo命令重新生成文章，部署成功后可以用Chrome浏览器的隐私标签来看下效果出来了没有。
