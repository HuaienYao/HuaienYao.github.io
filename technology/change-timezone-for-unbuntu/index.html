Ubuntu修改时区的简单方法-以修改UTC为JST为例

Written by Wyane with ♥ on June 12, 2018 in 技术记录

今天公司给了个课题，让我们查看AWS上的所有服务器是不是JST时区时间。 如果不是就需要修改为JST。 先用date命令来查看当前设置的时区。

网上查询了下Ubuntu的时区修改方法。

参照了Ubuntu设置时区和更新时间的教程。

解决方法

1、执行

tzselect

原文是推荐使用

sudo tzselect

但是本次没有用sudo也成功了

在出现的界面上选择你的时区。 这次我选的是Asia，Japan，Tokyo。这里只要输入对应选项前的号码就行了。

最后选择1）Yes 完成确认修改。

2、复制文件到 /etc/

sudo    cp /usr/share/zoneinfo/Asia/Tokyo /etc/localtime

3、命令更新时间*(本次我没有执行此更新命令，发现时区已经修改为JST了)

sudo ntpdate us.pool.ntp.org

最后用date来确认当前时区。

最后的最后，exit退出服务器
